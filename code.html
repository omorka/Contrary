The Code for Contrary

                        
                            //Get the string from the user
                            function getString()
                            {
                                document.getElementById("alert").classList.add("invisible");
                            
                                let userString = document.getElementById("userString").value;
                                let revString = reverseString(userString);
                                displayString(revString);
                            }
GetString() Method

The getString() method gets the string the user inputs from the HTML and stores it in a javascript value that will be used to reverse the string. It also adds the invisible class to the alert element to hide the alert box each time the user uses the app.

                        
                            function reverseString(userString)
                            {
                                let revString = [];
                            
                                //reverse a string
                                for(let i = userString.length - 1; i >= 0; i--)
                                {
                                    revString += userString[i];
                                }
                            
                                return revString;
                            }
ReverseString() Method

The reverseString() method has 1 parameter called userString and iterates from the end of the array to the beginning and adds the characters to the revString string array that then gets returned back to the control function to later be displayed.

                        
                            //Display the reversed string to the user
                            function displayString(revString)
                            {
                                //write to the page
                                document.getElementById("msg").innerHTML = `Your string is: ${revString}`;

                                //show alert box
                                document.getElementById("alert").classList.remove("invisible");
                            }
DisplayString() Method

The displayString() method takes in the string from the reverseString method and writes it to the HTML document. It also removes the invisible class from the alert element so that it is shown to the user.
